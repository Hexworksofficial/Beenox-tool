# Beenox-tool
Beenox Universal Extractor Pro

Overview:
This tool is designed to extract archives from multiple Spider-Man games across PC and console.
It provides a simple tab-based interface, automatic scanning, and live logging while extracting.

What the tool can do: 
- Extract archives from WOS Xbox (also SM console)
- Extract TASM 1 and TASM 2 game files
- Support Spider-Man 3 with two extraction methods
- Support Spider-Man EOT with two extraction methods
- Browse files and choose output folders easily
- Display real-time logs during extraction
- Scan PAK files using the built-in scanner
- Show system status while tasks are running

Updates / Changes / Improvements:
- Added multi-game support (WOS, TASM1, TASM2, SM3, SM EOT)
- Added Spider-Man 3 dual extraction system (Extract 1 / Extract 2)
- Added Spider-Man EOT dual extraction system
- Added RS Scanner tab for PAK scanning
- Improved UI layout with sidebar navigation
- Added live extraction logs per tab
- Added system status indicator (READY / PROCESSING)
- Improved error handling
- Improved browsing experience
- Added logo and branding
- Improved overall stability
- Cleaned up tab naming and buttons
- Removed unused features
- Optimized extraction flow
- Made everything more automated
